# jarvis

Персональная HTML-визитка Ирины Нагель.

Файлы в этом репозитории:
- `index.html` — главная страница для GitHub Pages
- `vizitka.html` — оригинальный файл визитки
- `photo.png` — фотография для аватара

## Как открыть
- Открой `index.html` в браузере
- После включения GitHub Pages страница будет доступна по адресу вида `https://irinanagel21-gif.github.io/jarvis/`
